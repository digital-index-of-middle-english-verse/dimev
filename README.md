# The Digital Index of Middle English Verse (DIMEV)

[![DOI](https://zenodo.org/badge/903481675.svg)](https://doi.org/10.5281/zenodo.14585960)
[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)

This repository contains data files and documentation for the *Digital Index of Middle English Verse* (DIMEV), an open-access bibliographic register of the surviving English-language verse from the period *c.* 1200--1525.
Contents are as follows:

- `data/` DIMEV data files
- `docs/` Components for the documentation
- `schemas/` XSD files for validation of data files
- `scripts/` Python scripts for formatting and validating XML

For additional detail see the [documentation](https://github.com/digital-index-of-middle-english-verse/dimev/releases/latest/download/documentation.pdf).
